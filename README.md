# netapp-openshift-console-protect

The open-source project **netapp-openshift-console-protect** has been designed to improve the user experience connecting Red Hat OpenShift with the NetApp portfolio.

netapp-openshift-console-protect deploys as a native Red Hat OpenShift console plugin enabling a graphical user interface to:  

- Accelerate the deployment of Trident and Trident Protect.
- Simplify the usage of Trident features like volumes, snapshots, mirrors, etcs.
- Provide application pattern deployment for fast prototyping up to production. 
- Enable dataset management for workload running in Red Hat OpenShift, OpenShift AI, and Developer Hub.

Documentation is available [here](DOCS.md).
